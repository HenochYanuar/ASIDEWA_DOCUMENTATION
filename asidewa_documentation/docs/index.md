# Selamat Datang Di ASI DEWA (Aplikasi Absensi Deteksi Wajah)

> Panduan lengkap untuk Pengguna dan Administrator.

Aplikasi Absensi Deteksi Wajah (ASI DEWA) adalah solusi modern dan efisien untuk manajemen kehadiran karyawan berbasis *webapp*. Dengan teknologi *Computer Vision* dan *Face Recognition*, kami memastikan proses absensi yang akurat, transparan, dan bebas dari kecurangan.

Dokumentasi ini dirancang untuk membantu Anda, baik sebagai karyawan yang melakukan absensi harian, maupun sebagai administrator yang mengelola data kehadiran.
